Module: public/js/snackbar

Provides functionality to display snackbar notifications for various messages. Offers methods for showing general, success, and error notifications.

Methods

(static) showSnackbar(message, type) → {void}

Displays a snackbar with the specified message and type. The snackbar shows the message, applies styling based on the type, and auto-hides after 5.5 seconds.
Parameters:
Name Type Description
message string The message to display.
type 'success' | 'error' | string The type of snackbar. Recognized types are 'success' and 'error'.
